# reviewSNS

## setup go dir
```
mkdir $HOME/go
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin
```

## setup lib

```
go get github.com/labstack/echo
go get github.com/pilu/fresh
go get github.com/go-sql-driver/mysql
```
